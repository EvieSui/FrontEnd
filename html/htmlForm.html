<input type="submit" value="">

提交按钮，用于提交表单

value属性指定按钮上的文字


<input type="reset">

重置按钮


<input type="button" value="">

按钮，通过JS绑定响应函数实现功能


<input type="text" name="" value="">

单行文本框，默认20个字符宽度


name属性指定内容描述名称

该属性是提供数据发送到服务器的必要信息，提交后信息会写入url以查询字符串的形式发送

查询字符串的格式:内容描述名称=文本框中的信息，提交多个信息以&链接

命名通常由后端决定


value属性指定文本框内的默认信息


<input type="password" name="">

密码框，默认字符被掩码


<input type="radio" name="" value="" checked="checked">

单选

name属性指定同组单选项需命名一致


value属性指定选择项各自的内容

该属性是需要用户选择的表单项提供数据发送到服务器的必要信息


checked="checked"指定默认选中的选项


<input type="checkbox" name="" value="" checked="checked">

复选框


<input type="file" name="" multiple>

文件

H5multiple属性实现选择多个文件


<input type="search" name="q" placeholder="" autocomplete="off" autofocus>

H5搜索，默认有清除内容功能，及搜索相似字符时显示提交历史

H5placeholder属性指定提示信息，无需删除即可键入

H5autocomplete属性获取历史提交信息选择后自动补全，可设置为on打开或off关闭

H5autofocus属性实现焦点状态为默认获取，打开和刷新页面后有效


<input type="email" name="" multiple>

H5邮箱，当提交的邮件地址格式不规范，提示相应规则，并阻断提交

H5增加multiple属性后，使用逗号分隔可提交多个邮件地址


<input type="url" name=""">

H5网址，网址必须包含http:


<input type="number" name="quantity" max="" min="" value="">

H5数量，只能输入数字和小数点


max属性限制可键入最大值

min属性限制可键入最小值

value属性设置默认值


<input type="range" name="" max="" min="" value="">

H5范围，滑块控件

默认值是中间值


<input type="color" name="" value="">

H5颜色选择器，默认值是#000


<input type="time" name="">

H5时间控件

Safari Not supported

H5日期控件

Safari Not supported

H5日期和时间控件

Firefox Not supported

Safari Not supported


<input type="month" name="">

H5年月控件

Firefox Not supported

Safari Not supported


<input type="week" name="">

H5年星期控件

Firefox Not supported

Safari Not supported


<input type="tel" name="" pattern="" required>

H5电话，实现在移动端启动数字键盘

H5pattern属性指定用于验证的正则表达式

H5required属性表示必须输入，未输入或输入格式不规范则弹出提示并阻止数据提交

Chrome Not supported

IE Not supported

Firefox Not supported

Opera Not supported


form="表单id"

H5该属性将表单外的表单元素与表单绑定，当表单提交时，当前被绑定的表单元素会一起提交
